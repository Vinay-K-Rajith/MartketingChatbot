Entab Chatbot Widget Demo

This page demonstrates how to embed the Entab chatbot widget using a single <script> tag.

Instructions:

  1. Start your Vite dev server (npm run dev).
  2. Start ngrok (ngrok http 5173).
  3. Copy your ngrok public URL and replace YOUR_NGROK_URL below.
  4. Open this HTML file in your browser.

You should see a floating chat button appear on the page.
